Class BooleanLocalizedFormatterBaseAbstract

Base class for boolean localized formatters. Booleans are unusual in that they aren't normally shown, and when they are shown, they need words meaningful to the user, not "true" and "false". "yes" and "no", "on" and "off", "enabled" and "disabled" are better. This base class lets you supply a list of cultures and the text that you want to use for true and false. If not supplied, it defaults to "true" and "false".
